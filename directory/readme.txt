SA Directory is a WordPress plugin that fetches data from an API and displays a directory of representatives on your website.

== Description ==

SA Directory allows you to display a directory of representatives on your WordPress site. The data is fetched from an external API, so you don't have to worry about updating the data manually. It includes a list of directories, a list of representatives for each directory, and a detailed page for each representative.

== Installation ==

1. Upload `sa-directory.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the your_url/directory display the directory.

== Frequently Asked Questions ==

= Where does the data come from? =

The data is fetched from an external API.

= How often is the data updated? =

The data is fetched from the API in real time whenever a user visits the directory page.
